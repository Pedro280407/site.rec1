Os Principais Comandos para Criar um Site

Comando HTML

: Declara o tipo de documento. : Envolve todo o conteúdo do site. : Contém metadados sobre o documento. : Define o título da página. <body>: Contém todo o conteúdo visível da página. <h1> a <h6>: Cabeçalhos, de h1 (mais importante) a h6 (menos importante). <p>: Parágrafo. <a>: Link para outra página ou recurso. <img>: Imagem. <div>: Contêiner genérico para agrupar outros elementos.</p> <img src="html.png" alt="Comando Html"> <h2>Comando CSS</h2> <p>color: Define a cor do texto. background-color: Define a cor de fundo de um elemento. font-size: Define o tamanho da fonte. margin: Define o espaço externo ao redor de um elemento. padding: Define o espaço interno dentro de um elemento. border: Define a borda ao redor de um elemento. display: Define como um elemento será exibido na página. text-align: Alinha o texto dentro de um elemento. width e height: Define a largura e a altura de um elemento. flexbox: Sistema de layout para criar layouts mais complexos.</p> <img src="css.png" alt="Comando Css"> </body> </html>
